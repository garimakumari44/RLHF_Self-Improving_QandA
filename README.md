## 🚀 RLHF Question–Answering System

A simple implementation of Reinforcement Learning from Human Feedback (RLHF) to improve the quality of answers generated by a language model.

### 📌 What this project does

This project builds a self-improving Q&A system:

Starts with a basic trained model
Learns what good answers look like
Improves responses using feedback
🔁 How it works
Question → Model Answer → Reward Score → Model Improves
