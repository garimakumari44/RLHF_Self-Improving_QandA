🚀 RLHF Question–Answering System

A simple implementation of Reinforcement Learning from Human Feedback (RLHF) to improve the quality of answers generated by a language model.

📌 What this project does

This project builds a self-improving Q&A system:

Starts with a basic trained model
Learns what good answers look like
Improves responses using feedback
🔁 How it works
Question → Model Answer → Reward Score → Model Improves
⚙️ Pipeline
SFT (Supervised Fine-Tuning)
Train the model on question–answer data
Reward Model
Train a model to score answers (good vs bad)
RLHF (PPO)
Improve the model using reward scores

📊 Dataset
SFT dataset → question + answer
Reward dataset → good answer vs bad answer
▶️ How to run
python train_sft.py
python train_reward_model.py
python train_ppo.py
🧠 Key idea

Instead of training once, the model keeps improving using feedback.
